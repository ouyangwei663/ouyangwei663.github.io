# 一、个人的见解

首先，不管说的深拷贝还是浅拷贝都只会存在于对象或者说是引用数据类型之中。基础类型没有深拷贝和浅拷贝这一说法。在引用数据类型中，我的理解是，我们看到的变量就是一个路标（指针），指向真正的存储这个信息的地址。如果我拷贝的是这个路标（指针），那就是浅拷贝，如果我拷贝的是这个储存信息的地址那么这个就深拷贝。 ​

# 二、为什么要区分深拷贝和浅拷贝

例如我想取两个完全不一样的变量 let a=['小明','小红','小绿','小爱']， 但是我现在想要一个b 变量，相对于a变量来说减少第一个元素，一般思路是这样的 let b=a; b.splice(0,1); console.log(a); //['小红','小绿','小爱'] console.log(b); //['小红','小绿','小爱'] 确实b是我们所要的结果，但是a并不是，因为b拷贝a的时候只是浅拷贝。但是b修改的时候确实修改了存储值，路标没有改变，但是存储值改变了，所以a当然改变了。所以这时候要用深拷贝。 ​

# 三、深拷贝的方法

（1）JSON方法 最简单的就是json拷贝

let a=['小明','小红','小绿','小爱']，
let b=JSON.parse(JSON.stringify(a));
b.splice(0,1);
console.log(a);  //['小明','小红','小绿','小爱']
console.log(b); //['小红','小绿','小爱']

注意：虽然这一方法简单，但是局限性也是最大的。首先属性上不能有函数，undefined 和symbol值.其次 不能拷贝对象原型链上的属性和方法。 （2）Object.assign（被赋值的对象，拷贝对象）

let a=['小明','小红','小绿','小爱']，
b=[];
Object.assign(b, a);
b.splice(0,1)
console.log(a);  //['小明','小红','小绿','小爱']
console.log(b); //['小红','小绿','小爱']

注意：这个拷贝可以运用于多数的深拷贝。但仅限于属性都是基本类型，不能再次嵌套引用类型，否则嵌套里面的属性也是浅拷贝 （3）递归拷贝（完全深拷贝）

    var obj = {            arr1 : [1,2,3],
            fn: function(){
                console.log('我是一个方法')
            },
            a : '我是普通属性'
        }
        
        // 现在我要把obj字面量创建里的属性深拷贝（ 属性值是引用类型也要深拷贝 ）
        function deepClone(obj){
            // 根据类型制造一个新的数组或对象 => 指向一个新的空间
                // 由于数组的typeof也是'object',所以用Array.isArray(obj)
            var new_obj = Array.isArray(obj) ? [] : {};
            // 首先判断obj的类型
                // 普通类型
            if( typeof obj != 'object' ){
                // 这里不能直接返回obj,不然就是浅拷贝的性质
                return  new_obj = obj
            }
                //引用类型
                    //数组
            if(obj instanceof Array ){
                for(i = 0; i < obj.length; i++ ){
                    new_obj[i] = obj[i];
                    if(typeof new_obj[i] == 'object'){
                        deepClone(new_obj[i])
                    }
                }
            }else{ //对象
                for (let key in obj) {
                    if (obj.hasOwnProperty(key)) {
                        // 对象中的数组和对象
                        if (typeof obj[key] == 'object') {
                            new_obj[key] = deepClone(obj[key]); 
                        }else{//对象中没有引用类型
                            new_obj[key] = obj[key]
                        }  
                    }
                }
            }
            return new_obj;
        }
        var deepClone = deepClone(obj);
        console.log(deepClone);
        // 测试是不是深拷贝
        obj.fn = '我改变了方法属性';
        console.log(obj); //{arr1: Array(3), fn: ƒ, a: "我是普通属性", c: "我新增了一个属性"}
        console.log(deepClone); // 还是 {arr1: Array(3), fn: ƒ, a: "我是普通属性"}

注意：这个拷贝属于完全拷贝。例子不是本人手写，摘自于简书。
